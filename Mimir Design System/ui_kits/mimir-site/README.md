# Mimir — marketing site UI kit

Bridges the app aesthetic into a public marketing voice. More breathing
room than the app, fewer chrome surfaces.

- `index.html` — interactive prototype: hero, features, how-it-works,
  pricing, footer.
- `Nav.jsx`, `Hero.jsx`, `Features.jsx`, `HowItWorks.jsx`, `Pricing.jsx`,
  `Footer.jsx`.

## Visual rules
- One strong character/vista plate per fold, used full-bleed, never as
  background pattern.
- Headline serif (Source Serif) — body sans (Inter).
- Display Njord reserved for the brand and one section opener.
- Plates are darkened toward ink on landing so type reads cleanly over them.
